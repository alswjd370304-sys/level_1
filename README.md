# 태그
## 다양한 문서 태그
* `h1~h6`:블록, 제목(대중소)
* `p`:블록, 내용(1줄부터 여러줄까지)
* `hr`:블록,구분선(디자인선 아님주의)
* `br`:인라인, 강제줄바꿈
* `em`:인라인, 특정문맥강조
* `strong`:인라인, 경고&할인 강조할 때
* `code`:인라인, 프로그래밍 코드 구분용
* `sup`, `sub`: 인라인 윗첨자, 아래첨자
* `q`, `blockquote`: 인라인(q) 짧은 인용, 블록(b)긴 인용문
* `s`, `del`: 인라인, s-유효하지 않은 내용, del-교체되거나 삭제된 내용

# 다양한 문서태그-2
## block tag
* `div`: 2개 이상의 자식을 묶는 그룹, class or id 필수
* `ul.li`: ul(비순차그룹), li(목록) 엄격한 부모-자식관계
* `ol.li`: ol(순차그룹), li(목록) 엄격한 부모-자식관계
* `dl,dt,dd`: 정의형그룹, 정의형제목, 정의내용
    * 검색키워드로 사용하지 않는 소제목과 내룔 `h-p`를 `dt-dd`로 대체가능
* `a`: 블록의 부모로 사용시 링크 (div 대체가능)
## inline tag
* `span`: 인라인 그룹, 2개 이상의 다른요소 묶기
* `a`: 블록의 부모로 사용시 링크 (div 대체가능)
### 비슷한 의미의 대체 태그들
* `h1~h6` vs `dt`
    * 중요도가 높고 키워드로서 사용되어야 한다면 h
    * 제목-내용이 바로 연결되고 중요도가 낮으며 키워드로서 사용가치가 낮은 때 dt
* `p` va `li`vs `dd` vs `span`
    * 제목 하단에 단순 다란으로 구성된 내용 표시시 p태그 사용
    * 2개 이상 연속된 목록 구성시 li
    * 중요도와 키워드가 낮은 제목 dt옆 내용 감쌀 땐 dd
    * 인라인으로 추가로 감싸야하는 내용이 있을 때 span
* `div` vs `ol` vs `ul` vs `a` vs `dl` vs `span`
    * div- 2개 이상의 블록, 인라인이 자식으로 있을 때 사용
    * ol- 순차가 있는 목록들을 묶을 때 사용
    * ul- 순차가 없는 목록들을 묶을 때 사용
    * a- 리읔로 선택되는 요소가 2개 이상있을 때 사용
    * dl- h3정도의 중요도가 있으면서 제목-내용 순으로 반복되는 요소들을 묶을 때 사용 / 중요도가 낮은 제목-내용이 있을 떄 그룹
    * span- 2개 이상의 인라인 요소들을 묶을 때 사용, 또는 변화할 수 있는 요소를 묶어줌.
